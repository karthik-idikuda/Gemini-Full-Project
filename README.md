# Gemini Full Project

Full-scale implementation of Google Gemini API integrations for search, content generation, and multi-modal AI capabilities.

## Features

- Text generation and summarization
- Image understanding and description
- Multi-turn conversational context
- Streaming response support
- Prompt template management

## Tech Stack

JavaScript, Gemini API, Node.js, Express

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
git clone https://github.com/karthik-idikuda/Gemini-Full-Project.git
cd Gemini-Full-Project
npm install
```

### Usage

```bash
npm start
```
